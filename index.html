Greenstock Online Ticket Sales

I set up a CentOS7 Server to run Attendize software to help The Higher Fellowship and IndyCann sell tickets for Greenstock, a music festival held in Indiana. This allowed the organizations to sell tickets online for the event while only having to pay fees to Paypal for credit card processing.

Virgil Davis Books

I created a website for Virgil Davis detailing him as an author, showcasing his book, The Soundbite Murders, as well as setting up a Fullcalendar.js calendar widget on an event page pointed at a Google calendar; allowing authorized users to add events to the website without modifying the code. All of this is hosted on a Digital Ocean CentOS7 server running Apache.
